# Barcha Digital

Website for Barcha Digital — a digital agency serving local businesses with custom web development, WordPress, Shopify, SEO, Google & social ads, branding, and AI-powered design.

Built with Next.js, Tailwind CSS, and GSAP.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Animation:** GSAP + Lenis smooth scroll
